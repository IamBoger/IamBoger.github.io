数组

数组：多个相同类型数据按一定顺序排列的组合，使用一个名字命名，并通过编号的方式对这些数据进行统一管理。

数组的特点：

​ 1、有序排列

​ 2、属于引用数据类型的变量。数组的元素，既可以是基本数据类型，也可以是引用数据类型

​ 3、创建数组对象会在内存中开辟一整块连续的空间

​ 4、数据的长度一旦确定，就不能修改

数组的分类：

​ 1、按照维数：一维数组、二维数组……

​ 2、按照元素类型：基本数据类型元素的数组、引用数据类型元素的数组

数组复制问题：

array1是一个数组，我们进行操作array2 = array1，是将array1的地址赋给了array2，这时候修改array2中的元素的值，array1元素的值也会发生改变，因为它们的地址是相同的，堆中实际上只有一个数组。所以 array2 = array1不算数组的复制。

注意：如果array1是一个一维数组，array2是一个二维数组，那么array1 = array2的操作是不允许的，因为虽然array1和array2实际上存的都是一个地址值，但是两个的变量类型不一样。赋值操作要求变量类型需要一致

Arrays工具类的使用

java.until.Arrays类即为操作数组的工具类，包含了用来操作数组的各种方法

方法举例：

boolean equals(int[] a, int[] b) — 判断两个数组是否相等

String toString(int[] a) — 输出数组信息

void fill(int[] a, int val) — 将指定值填充到数组之中（将数组中所有元素的值设置为指定值）

void sort(int[] a) — 对数组进行排序

int binarySearch(int[] a, int key) — 对排序后的数组进行二分法检索指定的值，返回该值的所在元素的下标，返回负数则意味着没找到指定值

调用方法的形式：Arrays.xxx(yyy)，xxx为方法名，yyy为参数列表
